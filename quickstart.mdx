## Prerequisites

<Callout kind="info">

Before you begin, ensure you have:
- A modern web browser like Chrome, Firefox, or Safari
- An email address for account verification
- Access to the [Anesthesia SmartSite](https://www.anesthesiasmartsite.com)

This quickstart guides you through account creation, dashboard navigation, and your first document upload.

</Callout>

## Create Account and Log In

Follow these steps to get your account set up quickly.

<Steps>
  <Step title="Sign Up" icon="user-plus">
    Visit `https://www.anesthesiasmartsite.com` and click **Sign Up**.

    Fill in your email, create a strong password, and complete the verification.

    ```bash
    # No code required - use the web form
    Email: your.email@example.com
    Password: your-secure-password
    ```
  </Step>

  <Step title="Verify Email" icon="mail">
    Check your inbox for the verification link from Anesthesia SmartSite.

    Click the link to activate your account.
  </Step>

  <Step title="Log In" icon="log-in">
    Return to the homepage and enter your credentials.

    You now have access to the dashboard.
  </Step>
</Steps>

## Upload Your First Document

Upload resources like presentation slides or resource guides using these methods.

<Tabs>
  <Tab title="Drag & Drop" icon="mouse-pointer">
    Open the **Uploads** section on the dashboard.

    Drag your file (e.g., `AI-in-Anesthesiology_Beyond-the-Hype_Brian-Pitts.pdf`) directly into the upload zone.

    ```

    bash
    # Supported formats: PDF, DOCX, MD, images
    ls ~/Downloads/AI-in-Anesthesiology_*.pdf
    # Drag to dashboard
    ```

  </Tab>

  <Tab title="File Picker" icon="upload">
    Click **Choose Files** in the Uploads area.

    Select your document and add metadata like title and tags (`VDD framework`, `anesthesiology AI`).

    <Callout kind="tip">
      Use tags for easy searching later.
    </Callout>
  </Tab>
</Tabs>

## Organize Projects and Resources

Structure your content for efficient management.

<Steps>
  <Step title="Create a Project" icon="plus">
    Click **New Project** and name it `36th Annual UC Davis Anesthesiology Update`.

    Add a description: `Napa 2026 resources using VDD framework`.
  </Step>

  <Step title="Add Resources" icon="file-plus">
    Move your uploaded document into the project folder.

    Create subfolders like `Slides`, `Quizzes`, and `Guides`.
  </Step>

  <Step title="Access Initial Resources" icon="external-link">
    Visit the built-in Resource Guide at [Resource Guide](https://www.anesthesiasmartsite.com/resource-guide.html).

    Explore the [Beyond the Hype Quiz](https://www.anesthesiasmartsite.com/ai/resources/beyond-the-hype-quiz.html).
  </Step>
</Steps>
