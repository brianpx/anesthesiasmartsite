## Upgrade Instructions

Follow these steps to update your Anesthesia SmartSite instance to the latest version.

<Steps>
  <Step title="Backup Current Site" icon="database">
    Create a full backup of your documentation and resources.
  </Step>
  <Step title="Check Release Notes" icon="book-open">
    Review changes in this changelog for any breaking updates.
  </Step>
  <Step title="Run Update Command" icon="download">
````bash
npm install anesthesiasmartsite@latest
````
  </Step>
  <Step title="Verify Installation" icon="check-circle">
    Test VDD framework and quiz functionality.
  </Step>
</Steps>

<Expandable title="Advanced Migration Guide" default-open="false">

For sites with custom integrations, update your config file:

````json
{
  "aiFramework": "vdd-v1.2",
  "resources": {
    "quizEnabled": true,
    "slidesUrl": "https://www.anesthesiasmartsite.com/AI-in-Anesthesiology_Beyond-the-Hype_Brian-Pitts.pdf"
  }
}
````

Restart your server after changes.

</Expandable>
